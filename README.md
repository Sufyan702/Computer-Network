 Project Overview
This project demonstrates a simple multi-department Small Campus Network using *Cisco Packet Tracer*.  
The network consists of three departments:
-  Admin
-  IT
- Library

Each department contains two PCs connected to a switch, and all switches connect to a central router for communication.

---

 Objectives
- Understand LAN communication and networking basics
- Configure IP addresses on end devices and router interfaces
- Test communication using the *ping* command
- Build and simulate a functional campus network topology

---

 Network Topology
Star-based topology with router as central connectivity device.
